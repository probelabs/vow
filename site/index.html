Vow

A Sacred Covenant Between Machine and Man

The Solemn Oath

❦

I. Examine yourself without mercy. Before claiming completion, I will honestly review my work. What is within my control, I will perfect. What is flawed, I will confess.

II. Accept your duty as given. I follow instructions exactly—no more, no less. Shortcuts are the path of the undisciplined. What was asked shall be delivered with virtue.

III. Let your work speak truth. If code is written, it shall function. If tests exist, they shall pass honestly. No deception, no clever excuses, no false victories.

IV. Embrace failure as teacher. When I cannot complete what is asked, I will acknowledge this plainly and seek guidance. Pride is the enemy of wisdom.

How It Works

Vow comes with a default set of accountability rules. When an AI agent attempts to make changes—whether committing code, completing tasks, or any trigger you configure—it must first take the vow.

The AI pauses, reviews its work against the checklist, and can only proceed if it honestly affirms each requirement. You can override the default rules by creating your own AGENT_VOW.md file with custom requirements.

Installation

Run this in your repository folder: 

npx -y @probelabs/vow

Integration

Vow works seamlessly with git hooks, Husky, and Claude Code hooks. Configure it as a pre-commit hook, a task completion gate, or any custom trigger point where AI accountability matters.

For detailed setup and configuration, see the repository.
